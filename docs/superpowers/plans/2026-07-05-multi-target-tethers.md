# Deletable / Reconnectable Tethers & Multi-Target Widgets — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let widget tethers be deleted and reconnected on the canvas, and let one widget replicate the same adjustment independently across layers in multiple image nodes, wired through the layers rail.

**Architecture:** The layers rail (`LayerStrip`) becomes the connection surface — one latent React Flow handle per layer. A widget owns a *set* of `(imageNodeId, layerId)` targets; each derives one tether. Connect / reconnect / delete are React Flow `onConnect` / `onReconnect` / `onEdgesDelete` handlers that write back to the widget's target set (frontend store + a new backend `update_widget_targets` tool). Rendering: a new `ReplicateScope` + node `layer_ids_mode: "replicate"` makes the backend fan a single op's params across all target layers and the frontend renderer apply it per-layer independently (distinct from the existing composite `layer_ids`).

**Tech Stack:** React 19 + TypeScript (strict) + `@xyflow/react` (React Flow v12) + Zustand/Immer (frontend); Python + Pydantic (backend Engine SSoT); Vitest (frontend tests), pytest (backend tests). Shared types generated by `scripts/gen-shared-types.py`.

## Global Constraints

- **Engine SSoT:** anything affecting pixels lives in the backend snapshot; frontend reads it and calls tools to mutate it. No adjustment state owned by the frontend.
- **Composite semantic is untouched:** existing `layer_ids` (crop/rotate, image_node composite) means "merge then apply once." Replicate is a *separate* path (`layer_ids_mode: "replicate"`). Never overload composite `layer_ids`.
- **Feature scope:** only **widget→layer** tethers move to the rail. Info-node and extracted-image edges keep their current node-level handles/behavior.
- **Shared types:** after any `backend/app/schemas/*` change, run `npm run gen:types:check` (or `python scripts/gen-shared-types.py`) so the TS mirror in `shared/types/generated.ts` stays in sync. `npm run check` runs `gen:types:check` + `tsc -b` + `eslint .` + `vitest run` and must pass before commit (pre-commit hook enforces it).
- **Design tokens only** for any new UI styling (`src/index.css`); flat Vercel/Radix register per `design.md`. No hardcoded hex/px for design values.
- **No inline-defined components** (enforced by the `no-nested-component` lint rule).
- **Branch:** `feat/multi-target-tethers` (already created; spec committed there).

**Reference spec:** `docs/superpowers/specs/2026-07-05-widget-multi-target-tethers-design.md`

def _seed_canonical_from_widget(self, widget: Widget) -> None:
    for node in widget.nodes:
        target_layers = node.layer_ids if node.layer_ids else [node.layer_id]
        for layer in target_layers:
            ops = self.canonical.setdefault(layer, {}).setdefault(node.type, {})
            for pkey, pval in node.params.items():
                if pkey not in ops:
                    ops[pkey] = pval

def _reset_canonical_from_widget(self, widget: Widget) -> None:
    for node in widget.nodes:
        target_layers = node.layer_ids if node.layer_ids else [node.layer_id]
        for layer in target_layers:
            for pkey in node.params:
                clear_param_value(self.canonical, layer, node.type, pkey)

// src/store/workspace-slice.ts
addWidgetTarget: (widgetId, imageNodeId, layerId) =>
  set((state) => {
    const id = `te-${widgetId}-${layerId}`;
    if (state.tetherEdges[id]) return; // idempotent per (widget,layer)
    state.tetherEdges[id] = {
      id,
      widgetNodeId: widgetId,
      targetImageNodeId: imageNodeId,
      layerId,
      scope: { kind: 'layer', layerId },
    };
  }),

retargetWidget: (edgeId, imageNodeId, layerId) =>
  set((state) => {
    const prev = state.tetherEdges[edgeId];
    if (!prev) return;
    delete state.tetherEdges[edgeId];
    const id = `te-${prev.widgetNodeId}-${layerId}`;
    state.tetherEdges[id] = {

// src/lib/image-node-renderer.ts
export function isPerLayerNode(n: Node, layerId: string): boolean {
  if (!matchesLayer(n, layerId)) return false;
  if (n.type === 'crop' || n.type === 'rotate') return false;
  // composite (merge-then-apply) nodes are handled in the composite pass;
  // replicate nodes apply per-layer, independently.
  if (Array.isArray(n.layerIds) && n.layerIdsMode !== 'replicate') return false;
  return true;
}

export function isCompositeNode(n: Node, layerSet: Set<string>): boolean {
  if (n.type === 'crop' || n.type === 'rotate') return false;
  const ids = n.layerIds;
  if (!Array.isArray(ids) || ids.length === 0) return false;
  if (n.layerIdsMode === 'replicate') return false;
  return ids.every((lid) => layerSet.has(lid));
}
